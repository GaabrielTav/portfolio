Barbearia App — Versão Mobile

Aplicativo desenvolvido para clientes realizarem agendamentos de forma rápida, acompanhar horários marcados e visualizar histórico de cortes. O foco principal foi simplicidade, clareza e experiência intuitiva.

UX/UI Mobile First Fluxo de Agendamento Prototipação

Problema

Clientes dependiam de mensagens via WhatsApp para marcar horário, gerando atrasos, conflitos de agenda e falta de organização.

Solução

Criação de uma interface mobile onde o cliente pode:

  • Selecionar barbeiro
  • Escolher tipo de serviço
  • Agendar horário disponível
  • Acompanhar status do atendimento
  • Visualizar histórico de cortes
