MoodFlix — Escolha Inteligente de Filmes

Aplicação desenvolvida para resolver um problema comum: gastar mais tempo escolhendo o filme do que assistindo. O sistema permite que o usuário cadastre seus próprios filmes, organize por categorias baseadas no humor e realize sorteios inteligentes.

HTML CSS JavaScript Lógica de Sorteio UX Centrada no Usuário

Problema

Usuários frequentemente passam muito tempo decidindo qual filme assistir. Plataformas de streaming oferecem muitas opções, mas pouca personalização baseada no humor específico do momento.

Solução

Desenvolvi um sistema onde o próprio usuário constrói sua base de filmes, organizando-os em categorias personalizadas como:

  • Filmes para rir
  • Filmes para refletir
  • Filmes leves
  • Filmes intensos
  • Filmes para assistir com amigos

A partir dessas categorias, o usuário pode:

  • Visualizar a lista completa
  • Filtrar por humor específico
  • Sortear um filme aleatoriamente
  • Gerar uma lista de filmes sorteados em ordem

Funcionalidades Principais

  • Cadastro de filmes personalizado
  • Criação de categorias baseadas no humor
  • Sistema de sorteio inteligente
  • Histórico de filmes sorteados
  • Interface simples e objetiva

Diferencial do Projeto

Diferente de um simples sorteador, o MoodFlix trabalha com contexto emocional. O usuário define o tipo de experiência que deseja ter e o sistema respeita essa intenção no momento da escolha.
